# 環境設定
このプログラムを実行するためには、以下のライブラリが必要です。

- beautifulsoup4
- requests
これらのライブラリをインストールするには、以下のコマンドを実行してください。

```
pip install beautifulsoup4 requests
```

## 実行方法

タグ名はサンプルです。任意のものを入力してください。

```
python main.py "atsuko_(blue_archive)"
```
