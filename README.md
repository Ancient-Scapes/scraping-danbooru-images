# 環境設定

```
pip install -r requirements.txt
```

## 実行方法

タグ名はサンプルです。任意のものを入力してください。

```
python main.py "atsuko_(blue_archive)"
```

```
[INFO] 1ページ目を開いています...
[INFO] 1個目の画像(*****.jpg)をダウンロードしました。
[INFO] 2個目の画像(*****.png)をダウンロードしました。
```

ダウンロードが完了したら、以下のようなメッセージが表示されます。

```
[INFO] 全***個の画像をダウンロードしました。
```
